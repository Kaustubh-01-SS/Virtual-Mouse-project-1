# Virtual-Mouse-project-1
First install Dependencies

AI Virtual Mouse Project

Controls:
 Index STRAIGHT and all other fingers down  → MOVE CURSOR.
 
 Only INDEX+MIDDLE finger UP → LEFT CLICK.
 
 Only MIDDLE+RING finger UP → RIGHT CLICK.
 
 Index + Middle + Thumb STRAIGHT → DRAG MODE (Hold Click).
 
 All 5 Fingers UP + Move hand UP → SELECT NEXT APP.
 
 All 5 Fingers UP + Move hand DOWN → SELECT PREVIOUS APP.
 
 All 5 Finger tips joint together + Wave UP → SCROLL DOWN.
 
 All 5 Finger tips joint together + Wave DOWN → SCROLL UP.
 
 Press ESC to EXIT.
